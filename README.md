# entity-log
Trained Bi-LSTM model that transforms raw logs into rich structured information.

## Pretrained model
Fetch the pretrained model by running: <br>
```
1. pip install gdown
2. gdown https://drive.google.com/uc?id=1oz8wl6MbXFFkpSh83sSMIM0dIQ37ZYre
```
